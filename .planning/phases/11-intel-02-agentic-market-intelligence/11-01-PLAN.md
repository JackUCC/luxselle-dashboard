**MarketIntelMonitorService.ts changes:**

Add constant at top of file (after imports):
```typescript
const COST_PER_CALL_EUR = 0.004
const STANDARD_ANALYSE_CALL_COUNT = 3  // expandQuery + searchMarket + extractStructuredJson
```

In `executeRun()`, in the try block after `const enrichedResult = ...`, before calling `snapshotRepo.upsertByKey()`:
```typescript
const aiUsage = {
  callCount: STANDARD_ANALYSE_CALL_COUNT,
  provider: enrichedResult.provider,
  estimatedCostEur: Math.round(STANDARD_ANALYSE_CALL_COUNT * COST_PER_CALL_EUR * 1000) / 1000,
}
```

When calling `runRepo.set(runningRun.id, { status: 'succeeded', ... })`, include `aiUsage` in the set payload:
```typescript
const completedRun = await this.runRepo.set(runningRun.id, {
  status: 'succeeded',
  completedAt,
  snapshotId: snapshot.id,
  result: enrichedResult,
  aiUsage,
  updatedAt: completedAt,
})
```

The catch block (`status: 'failed'`) does NOT include aiUsage — this is intentional (partial runs get no cost attribution).
